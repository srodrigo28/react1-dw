### Passos para Criação do projeto

* 1. Criação do projeto
```
npm create vite@latest
```
* 2. Entrar na pasta do projeto
```
cd react2
```
* 3. Instalar as depedências
```
npm i
```
* 4. Executar o projeto
```
npm run dev
```
